# Event Registration App

## 🎯 Goal
This project is built to practice React and TypeScript fundamentals by implementing a simple event registration system.

## 🧩 Features
- View list of events
- Select an event
- Register for an event (name + email)
- View list of participants

---

## 🛠 Tech Stack
- React
- TypeScript
- Vite
